# lamp

### Запуск прокси сервера

```bash
php lamp.php up proxy
```

### Создание серверов

```bash
php lamp.php up [имя сервера]
```

### Остановка сервера

```bash
php lamp.php stop [имя сервера]
```

### Удаление сервера

```bash
php lamp.php delete [имя сервера]
```

Список серверов

```bash
php lamp.php list
```
